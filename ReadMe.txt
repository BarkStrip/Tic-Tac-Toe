Tic-Tac-Toe:
Main Menu
-New Game - Starts a new game
-Continue Game - Continues a game already in progress, where no winner has been determined yet
-Settings - Allows users to change their usernames and play against the computer
-Exit - Exits the program

In-Game
-Main Menu - Goes to the main menu, without resetting the game.
-Restart Game - Restarts the current game

Note:
-If the user enables the setting to play against the computer, while a game is in progress, the game resets
